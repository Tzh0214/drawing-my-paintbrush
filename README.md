My Paintbrush 绘图应用
一、简介
My Paintbrush 是一款功能丰富的绘图应用，支持多种绘图工具、图层管理、对象编辑等功能，为用户提供便捷且灵活的数字绘图体验。
二、功能介绍
（一）绘图工具
铅笔工具：可自由绘制线条，通过右侧工具选项面板可调整画笔颜色和大小。
直线工具：用于绘制直线，能设置线条颜色和粗细。
矩形工具：绘制矩形，可选择是否填充以及填充颜色、边框颜色和粗细。
圆形工具：绘制圆形或椭圆形，同样支持填充和边框设置。
多边形工具：绘制多边形，点击画布添加顶点，双击可闭合多边形，可设置边框和填充属性。
文本工具：添加文字，点击画布会弹出对话框，可设置文字内容、字体、字号和颜色。
橡皮擦工具：有 “局部” 和 “对象” 两种模式，“局部” 模式可擦除画布局部区域，“对象” 模式可直接删除选中的图形对象。
油漆桶工具：对闭合区域进行颜色填充。
（二）图层管理
图层操作：可新建、复制、删除图层，还能调整图层顺序（上移、下移图层）。
图层属性：在右侧图层面板，可设置图层的可见性（显示 / 隐藏图层）和不透明度，通过滑块调整不透明度数值。
图层选择：点击图层列表中的图层名称，可选中该图层，后续绘图操作将在该图层上进行。
（三）对象编辑
选择工具：用于选择画布上的图形对象，选中后可对对象进行移动、旋转、缩放等操作。
移动对象：选中对象后，按住鼠标左键拖动即可移动对象位置。
旋转对象：选中对象后，拖动旋转控制柄可旋转对象，也可通过相关操作精确设置旋转角度。
缩放对象：拖动对象的缩放控制柄，可对对象进行缩放操作，调整对象大小。
（四）项目与导出
项目保存与打开：可将当前绘图项目保存为特定格式文件，方便后续打开继续编辑；也能打开已保存的项目文件。
图片导出：支持将绘图内容导出为 PNG、JPEG、BMP 等格式的图片文件。
（五）辅助功能
网格显示：可开启或关闭网格显示，辅助绘图时的位置对齐。
撤销操作：通过快捷键 Ctrl+Z 或右侧工具选项面板的 “撤销” 按钮，撤销上一步操作。
三、复杂功能使用说明
（一）多边形工具使用
点击左侧工具栏的 “多边形工具” 按钮。
在画布上依次点击，添加多边形的顶点，此时会显示临时的多边形边。
当要闭合多边形时，双击画布即可完成多边形绘制。
若要调整多边形的边框或填充，可在右侧工具选项面板设置相应的颜色和粗细等参数。
（二）文本工具使用
点击左侧工具栏的 “文本工具” 按钮。
在画布想要添加文字的位置点击，会弹出 “文本工具” 对话框。
在 “文本内容” 输入框中输入要添加的文字。
从 “字体” 下拉菜单中选择合适的字体，在 “字号” 输入框中输入或选择字号大小。
点击 “确定”，文字会添加到画布上，若需要调整文字位置，可使用选择工具移动。
（三）图层管理
新建图层：点击右侧图层面板的 “新建图层” 按钮，会创建一个新的图层，后续绘图将在该新图层进行。
调整图层顺序：选中要调整顺序的图层，点击 “上移图层” 或 “下移图层” 按钮，可改变图层在堆叠顺序中的位置，图层顺序会影响图形的显示层次。
设置图层不透明度：在图层面板中，拖动对应图层的不透明度滑块，可调整该图层的不透明度，实现半透明等效果。
（四）对象旋转与缩放
使用选择工具选中要编辑的对象，此时对象周围会出现旋转和缩放控制柄。
旋转对象：将鼠标指针移到旋转控制柄（通常是对象外的圆形控制柄）上，按住鼠标左键拖动，即可旋转对象；也可通过相关精确设置功能，输入具体的旋转角度值。
缩放对象：将鼠标指针移到缩放控制柄（对象四周的方形控制柄）上，按住鼠标左键拖动，可对对象进行缩放，调整对象的宽度和高度。
